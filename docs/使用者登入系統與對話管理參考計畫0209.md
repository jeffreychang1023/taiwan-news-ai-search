# 實作計畫：B2B 使用者登入 + 對話管理系統

## Context

目前系統以單機開發為主，user ID 全靠 `TEMP_USER_ID = 'demo_user_001'` hardcode（7 處使用），搜尋歷史全存 localStorage（`taiwanNewsSavedSessions`），API 端點以 query param 傳 `user_id`（不安全）。要轉為 B2B 線上服務，需要：
1. 組織制的 Email/Password 登入系統
2. Server-side 對話/偏好管理，取代 localStorage

---

## 資料庫選擇：Neon PostgreSQL（擴展現有）

| 選項 | 優點 | 缺點 | 結論 |
|------|------|------|------|
| **Neon PostgreSQL** | 已在用（analytics）、psycopg 已裝、RLS 支援多租戶、零新基建 | 需自建 auth 邏輯 | **推薦** |
| Supabase | 內建 Auth + RLS | 額外服務、供應商鎖定 | 過度 |
| AWS RDS / GCP Cloud SQL | 企業級 | 早期階段太貴 | 太早 |
| MongoDB Atlas | 彈性 schema | 關聯型 org/user 資料不適合 | 不合 |
| Turso (libSQL) | 便宜 edge | 無 RLS、工具少 | 不合 |

**理由**：已有 `ANALYTICS_DATABASE_URL` 連 Neon，`analytics_db.py` 有 SQLite/PostgreSQL 雙支援 pattern 可複用。同一個 DB 加新 table 即可。

## Part 1：使用者登入系統

### Phase 1A — DB Schema（Auth Tables）

```sql
-- 組織
CREATE TABLE organizations (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    name VARCHAR(255) NOT NULL,
    slug VARCHAR(100) NOT NULL UNIQUE,
    plan VARCHAR(50) DEFAULT 'free',
    max_members INT DEFAULT 10,
    settings JSONB DEFAULT '{}',
    created_at TIMESTAMPTZ DEFAULT NOW(),
    updated_at TIMESTAMPTZ DEFAULT NOW(),
    is_active BOOLEAN DEFAULT TRUE
);

-- 使用者
CREATE TABLE users (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    email VARCHAR(320) NOT NULL UNIQUE,
    password_hash VARCHAR(255) NOT NULL,
    name VARCHAR(255) NOT NULL,
    email_verified BOOLEAN DEFAULT FALSE,
    email_verification_token VARCHAR(255),
    password_reset_token VARCHAR(255),
    password_reset_expires TIMESTAMPTZ,
    last_login TIMESTAMPTZ,
    created_at TIMESTAMPTZ DEFAULT NOW(),
    is_active BOOLEAN DEFAULT TRUE
);

-- 組織成員（多對多 + 角色）
CREATE TABLE org_memberships (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
    org_id UUID NOT NULL REFERENCES organizations(id) ON DELETE CASCADE,
    role VARCHAR(50) NOT NULL DEFAULT 'member',  -- admin, member, viewer
    invited_by UUID REFERENCES users(id),
    status VARCHAR(20) DEFAULT 'pending',  -- pending, active, suspended
    accepted_at TIMESTAMPTZ,
    UNIQUE(user_id, org_id)
);

-- 邀請（使用者尚未註冊時）
CREATE TABLE invitations (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    org_id UUID NOT NULL REFERENCES organizations(id) ON DELETE CASCADE,
    email VARCHAR(320) NOT NULL,
    role VARCHAR(50) DEFAULT 'member',
    invited_by UUID NOT NULL REFERENCES users(id),
    token VARCHAR(255) NOT NULL UNIQUE,
    expires_at TIMESTAMPTZ NOT NULL,
    accepted_at TIMESTAMPTZ,
    created_at TIMESTAMPTZ DEFAULT NOW()
);

-- Refresh Tokens
CREATE TABLE refresh_tokens (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
    token_hash VARCHAR(255) NOT NULL UNIQUE,
    expires_at TIMESTAMPTZ NOT NULL,
    created_at TIMESTAMPTZ DEFAULT NOW(),
    revoked_at TIMESTAMPTZ
);
```

### Phase 1B — 後端 Auth 模組

**新檔案：`code/python/auth/auth_db.py`**
- 複用 `core/analytics_db.py` 的雙 DB pattern（本地 SQLite / Production PostgreSQL）

**新檔案：`code/python/auth/auth_service.py`**

核心方法：

| 方法 | 說明 |
|------|------|
| `register_user(email, password, name)` | bcrypt hash → 建立 user |
| `login(email, password)` | 驗證 bcrypt → 回傳 JWT access token + refresh token |
| `refresh_token(token)` | 刷新 access token |
| `logout(token)` | 撤銷 refresh token |
| `create_organization(name, admin_user_id)` | 建立 org + admin membership |
| `invite_member(org_id, email, role, invited_by)` | 建立邀請 |
| `accept_invitation(token, user_id)` | 接受邀請 → 建立 membership |
| `list_org_members(org_id)` | 列出成員 |
| `remove_member(org_id, user_id, acting_user_id)` | 移除成員（admin only） |
| `forgot_password(email)` / `reset_password(token, new_pw)` | 密碼重設 |

**Token 策略：**
- Access token：15 分鐘，存 JS 變數，via `Authorization: Bearer` header
- Refresh token：7 天，存 HttpOnly cookie（path `/api/auth/refresh`）
- JWT payload：`{user_id, email, name, org_id, role, exp, iat, type}`

**依賴：** `bcrypt`, `PyJWT`（已裝）

### Phase 2A — Session 儲存 Schema

```sql
-- 搜尋 Session（取代 localStorage taiwanNewsSavedSessions）
CREATE TABLE search_sessions (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
    org_id UUID NOT NULL REFERENCES organizations(id),
    title VARCHAR(500),
    conversation_history JSONB DEFAULT '[]',
    session_history JSONB DEFAULT '[]',
    chat_history JSONB DEFAULT '[]',
    accumulated_articles JSONB DEFAULT '[]',
    pinned_messages JSONB DEFAULT '[]',
    pinned_news_cards JSONB DEFAULT '[]',
    research_report JSONB DEFAULT '{}',

    admin_note TEXT,              -- 管理員標記（供系統改善用）
    is_archived BOOLEAN DEFAULT FALSE,
    created_at TIMESTAMPTZ DEFAULT NOW(),
    updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- 使用者偏好（取代 localStorage source_folders, file_folders, selected_files）
CREATE TABLE user_preferences (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
    org_id UUID NOT NULL REFERENCES organizations(id),
    preference_key VARCHAR(100) NOT NULL,  -- 'source_folders', 'file_folders', 'selected_files'
    preference_value JSONB NOT NULL,
    updated_at TIMESTAMPTZ DEFAULT NOW(),
    UNIQUE(user_id, org_id, preference_key)
);

CREATE INDEX idx_sessions_user_org ON search_sessions(user_id, org_id);
CREATE INDEX idx_sessions_updated ON search_sessions(updated_at DESC);
CREATE INDEX idx_prefs_user_org ON user_preferences(user_id, org_id);

ALTER TABLE search_sessions ENABLE ROW LEVEL SECURITY;
ALTER TABLE user_preferences ENABLE ROW LEVEL SECURITY;
```

- 目前 localStorage 同步存、無延遲

## 驗證方式

### Sprint 1 完成後
```bash
# 註冊
curl -X POST http://localhost:8080/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{"email":"admin@test.com","password":"Test1234!","name":"Admin"}'

# 登入
curl -X POST http://localhost:8080/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"admin@test.com","password":"Test1234!"}'
# → 回傳 access_token + Set-Cookie refresh token

# 建立組織
curl -X POST http://localhost:8080/api/org \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"name":"Test Corp"}'
```
