# Plan B1: Subprocess Per Crawler

## Context

Full scan crawlers currently share a single Python process and event loop. Despite `run_in_executor` optimization (2.75x improvement), BeautifulSoup parsing still bottlenecks due to GIL contention across crawlers. Each crawler achieves ~1.1 req/s instead of theoretical ~8 req/s.

**Goal**: Launch each crawler as a separate Python subprocess for true GIL isolation and independent event loops. This should allow each crawler to reach its maximum throughput independently.

**Constraint**: 100% coverage is the priority. No skipping candidate URLs or any shortcut that could miss articles.

## Architecture Overview

```
Dashboard (parent process)
  ├── asyncio.create_subprocess_exec("python", "subprocess_runner.py", ...)
  │     └── Crawler A (own process, own event loop, own GIL)
  ├── asyncio.create_subprocess_exec("python", "subprocess_runner.py", ...)
  │     └── Crawler B (own process, own event loop, own GIL)
  └── reads stdout (JSON lines) for progress, checks process exit for completion
```

**IPC Protocol**: JSON lines on subprocess stdout
**Stop Mechanism**: Signal file + process.terminate() fallback
**SQLite**: Already WAL mode, safe for multi-process concurrent writes

### 1. NEW: `code/python/crawler/subprocess_runner.py`

Entry point for subprocess crawlers. Receives params via CLI args, outputs JSON lines to stdout.

```python
# Usage: python -m crawler.subprocess_runner --params '{"source":"ltn","mode":"full_scan",...}'
#        --task-id "fullscan_ltn_5_1234567890" --signal-dir "data/crawler/signals"

import argparse, asyncio, json, sys, time, logging
from pathlib import Path

# Redirect all logging to stderr (stdout reserved for JSON protocol)
logging.basicConfig(stream=sys.stderr, level=logging.INFO)

def _send(msg: dict):
    """Write JSON line to stdout (parent reads this)"""
    print(json.dumps(msg, ensure_ascii=False), flush=True)

async def main(params, task_id, signal_dir):
    from crawler.parsers.factory import CrawlerFactory
    from crawler.core.engine import CrawlerEngine

    source = params["source"]
    parser = CrawlerFactory.get_parser(source)

    signal_file = Path(signal_dir) / f".stop_{task_id}"

    def on_progress(stats):
        _send({"type": "progress", "stats": stats})

    def check_stop():
        return signal_file.exists()

    engine = CrawlerEngine(
        parser=parser,
        auto_save=True,
        progress_callback=on_progress,
        chunk_size=params.get("chunk_size", 0),
        chunk_by_month=params.get("chunk_by_month", False),
        stop_check=check_stop,  # NEW parameter
    )

    # Run based on mode
    mode = params.get("mode")
    if mode == "full_scan":
        result = await engine.run_full_scan(...)
    elif mode == "auto":
        result = await engine.run_auto(...)
    # ... other modes

    await engine.close()
    _send({"type": "completed", "stats": result})

if __name__ == "__main__":
    parser = argparse.ArgumentParser()
    parser.add_argument("--params", required=True)
    parser.add_argument("--task-id", required=True)
    parser.add_argument("--signal-dir", required=True)
    args = parser.parse_args()

    params = json.loads(args.params)
    asyncio.run(main(params, args.task_id, args.signal_dir))
```

Key design:
- All logging goes to **stderr** (redirected to log file by parent)
- Only JSON protocol goes to **stdout**
- `stop_check` callback: engine checks signal file periodically
- Exit code 0 = normal completion, non-zero = error
- Sends `{"type": "completed", "stats": {...}}` on exit

**New method `_run_crawler_subprocess()`**:
```python
async def _run_crawler_subprocess(self, task: CrawlerTask, params: Dict) -> None:
    """Launch crawler as subprocess and read JSON lines from stdout."""
    import sys

    signal_dir = Path(self.SIGNAL_DIR)
    signal_dir.mkdir(parents=True, exist_ok=True)

    params_json = json.dumps({**params, "source": task.source})

    # Launch subprocess
    # Must run from code/python/ directory for imports to work
    code_dir = str(Path(__file__).parent.parent)
    proc = await asyncio.create_subprocess_exec(
        sys.executable, "-m", "crawler.subprocess_runner",
        "--params", params_json,
        "--task-id", task.task_id,
        "--signal-dir", str(signal_dir),
        stdout=asyncio.subprocess.PIPE,
        stderr=asyncio.subprocess.PIPE,  # or redirect to log file
        cwd=code_dir,
    )
    task._process = proc

    # Read stdout JSON lines
    try:
        async for line in proc.stdout:
            text = line.decode("utf-8").strip()
            if not text:
                continue
            try:
                msg = json.loads(text)
            except json.JSONDecodeError:
                continue

            if msg["type"] == "progress":
                stats = msg["stats"]
                task.stats = stats
                task.progress = stats.get("progress", stats.get("success", 0) + stats.get("skipped", 0))
                task.total = stats.get("total", task.total)
                if "last_scanned_id" in stats:
                    task.last_scanned_id = stats["last_scanned_id"]
                if "last_scanned_date" in stats:
                    task.last_scanned_date = stats["last_scanned_date"]
                asyncio.create_task(self._broadcast_status(task))

            elif msg["type"] == "completed":
                task.stats = msg["stats"]
                # early_stopped check
                if msg["stats"].get("early_stopped"):
                    task.status = CrawlerTaskStatus.EARLY_STOPPED
                    task.early_stop_reason = msg["stats"].get("early_stop_reason")
                else:
                    task.status = CrawlerTaskStatus.COMPLETED
                task.finished_at = time.time()

        # Wait for process exit
        await proc.wait()

        if proc.returncode != 0 and task.status == CrawlerTaskStatus.RUNNING:
            stderr_output = await proc.stderr.read()
            task.status = CrawlerTaskStatus.FAILED
            task.error = stderr_output.decode("utf-8")[-500:]  # last 500 chars
            task.finished_at = time.time()

    except asyncio.CancelledError:
        # stop_crawler was called
        task.status = CrawlerTaskStatus.STOPPING
        task.finished_at = time.time()

    except Exception as e:
        task.status = CrawlerTaskStatus.FAILED
        task.error = str(e)
        task.finished_at = time.time()

    finally:
        # Clean up signal file
        signal_file = signal_dir / f".stop_{task.task_id}"
        if signal_file.exists():
            signal_file.unlink()

        await self._broadcast_status(task)
```

## IPC Protocol

```jsonl
{"type": "progress", "stats": {"total": 1000, "success": 50, "failed": 2, "skipped": 10, "not_found": 938, "last_scanned_id": 7800050}}

{"type": "completed", "stats": {"total": 1000, "success": 146, "failed": 8, "skipped": 30, "not_found": 816, "early_stopped": false}}
```

## Stop Mechanism

1. Dashboard creates signal file: `data/crawler/signals/.stop_{task_id}`
2. Engine checks in `_report_progress()` (called every ~1s)
3. If signal file exists -> `raise asyncio.CancelledError`
4. Engine's `run_full_scan` catches CancelledError, returns partial stats
5. Subprocess sends `{"type": "completed", "stats": {...}}` and exits
6. If subprocess doesn't exit within 10s, parent calls `proc.terminate()`
7. Parent cleans up signal file
