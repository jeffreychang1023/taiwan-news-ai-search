## 二、服務說明

本服務為一繁體中文新聞搜尋與 AI 分析平台，提供以下核心功能：

- **語意搜尋**：以自然語言查詢臺灣主要媒體新聞
- **深度研究分析**：多重 AI Agent 對查詢主題進行多角度分析
- **引用溯源**：所有 AI 分析結果附有原始新聞來源連結
- **企業 API 存取**（付費企業方案）：以程式方式整合本服務功能

本服務之目標使用者為知識工作者，包含記者、研究人員、公關、商業分析師及政策工作者。
