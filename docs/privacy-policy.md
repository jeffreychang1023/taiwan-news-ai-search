## 一、總則

讀豹資訊有限公司（以下簡稱「本公司」或「我們」）深知個人資料保護的重要性。本隱私權政策說明我們在您使用「讀豹」新聞搜尋服務（網址：https://twdubao.com，以下簡稱「本服務」）時，如何蒐集、處理、利用及保護您的個人資料。

本政策依據中華民國《個人資料保護法》（以下簡稱「個資法」）及相關法規制定。

**本政策適用範圍**：本服務所有功能，包含搜尋、深度研究分析、帳號管理及相關衍生功能。

**不適用範圍**：本政策不適用於本服務所連結之外部網站或第三方服務，該等外部服務之隱私保護請參閱其各自之隱私權政策。

使用本服務即表示您已閱讀、瞭解並同意本政策之內容。若您不同意，請立即停止使用本服務並申請刪除帳號。

## 三、蒐集的個人資料種類與不提供之影響

### 3.1 帳號必要資料（不提供則無法使用服務）

| 資料項目 | 用途 | 不提供之影響 |
|----------|------|--------------|
| 電子郵件地址 | 帳號識別、通知、密碼重設 | **無法建立帳號，無法使用本服務** |
| 密碼 | 身份驗證，以 bcrypt 雜湊加密儲存 | **無法登入** |

### 3.2 使用行為資料（服務運作自動產生）

您使用本服務時，系統自動記錄下列資料，此為提供服務之必要技術資料：

- **搜尋查詢內容**：您輸入的搜尋字詞，用於執行搜尋及改善搜尋品質
- **搜尋結果互動**：點擊的文章連結、引用的內容，用於排序模型改善
- **Session 資訊**：對話 ID、建立時間、最後活動時間，用於維持對話上下文
- **查詢分析資料**：查詢類型、功能使用情形、回應時間，用於服務品質監控

> **重要**：您輸入的搜尋查詢內容將傳送至第三方 AI 語言模型服務（詳見第八條）以產生搜尋分析結果。請勿於搜尋欄位輸入敏感個人資料。

### 3.3 技術日誌（安全與合規必要）

- **IP 位址**：安全審計、異常偵測、防止濫用
- **存取日誌**：請求時間、API 端點、HTTP 狀態碼，用於故障排除與安全監控
- **裝置與瀏覽器資訊**：User-Agent 字串，用於技術相容性與異常偵測

### 3.4 組織帳號資料（B2B 方案，適用組織成員）

若您透過組織 Bootstrap Token 啟用帳號，另記錄：

- 所屬組織識別碼（`org_id`）
- 帳號啟用方式（Bootstrap Token 來源）
- 帳號角色與權限層級

## 五、Cookie 與認證 Token 使用說明

### 5.1 本服務使用之 Cookie 與 Token

| 類型 | 說明 | 儲存方式 | 存活期限 |
|------|------|----------|----------|
| JWT Access Token | 短效期身份驗證令牌，用於 API 請求授權 | 瀏覽器記憶體（localStorage） | 15 分鐘 |
| JWT Refresh Token | 長效期令牌，用於自動更新 Access Token | HttpOnly Cookie（無法被 JS 讀取） | 7 天 |
| Session Cookie | 維持搜尋對話狀態 | Cookie | Session 期間 |

### 5.2 Cookie 管理

您可透過瀏覽器設定拒絕或刪除 Cookie。**請注意**：停用 Cookie 將導致無法維持登入狀態，本服務將無法正常使用。

本服務**不使用**廣告追蹤 Cookie 或跨網站追蹤技術。

## 六、個人資料之保存期限

| 資料類型 | 保存期限 | 說明 |
|----------|----------|------|
| 帳號基本資料（Email） | 帳號存續期間 + 30 天 | 帳號刪除後 30 天內清除 |

| 組織帳號稽核紀錄 | 2 年 | B2B 合規要求 |

保存期限屆滿後，資料將以不可回復方式刪除或匿名化處理。

## 七、個人資料之安全措施

### 7.1 技術措施

- 密碼以 bcrypt 演算法雜湊儲存，不儲存明文密碼
- 全站傳輸採 HTTPS/TLS 加密
- JWT Refresh Token 儲存於 HttpOnly Cookie，防止 XSS 竊取
- 資料庫存取採最小權限原則
- 並發請求限制（防止暴力攻擊）

### 7.2 組織措施

- 定期安全審計
- 員工個資存取紀錄留存
- 委外服務商簽署保密協議

### 7.3 資料外洩通知

若發生個人資料外洩事件，本公司將：

1. 於發現後 **72 小時內**評估影響範圍
2. 若影響您的權益，於合理時間內以電子郵件通知受影響用戶
3. 依個資法規定向主管機關報告

境外傳輸依個資法第 21 條規定，由本公司確保境外接收方具備足夠個資保護水準，或訂有契約約束。

## 九、自動化決策說明

本服務使用自動化系統處理您的搜尋請求，包含：

- **查詢分析**：自動判斷搜尋意圖、拆解複雜查詢
- **內容排序**：以演算法（XGBoost + MMR）對搜尋結果排序
- **安全偵測**：自動偵測並封鎖惡意查詢（Prompt Injection、Jailbreak）

上述自動化處理**不涉及**對您個人的法律效力決定或重大影響評估。如對自動化處理結果有異議，可聯絡本公司請求人工審查。

## 十、您的個資權利及行使方式

依據《個人資料保護法》第 3 條，您享有以下權利：

| 權利 | 說明 | 行使方式 |
|------|------|----------|
| 查詢或閱覽 | 查看本公司持有哪些您的個人資料 | 電子郵件申請 |
| 製給複製本 | 取得個人資料副本（資料可攜） | 電子郵件申請 |
| 補充或更正 | 更正不正確或不完整的資料 | 登入帳號設定或電子郵件申請 |
| 停止蒐集、處理或利用 | 要求停止處理特定資料 | 電子郵件申請 |
| 刪除 | 要求刪除帳號及相關個人資料 | 電子郵件申請，或帳號設定頁 |

**回應時限**：本公司於收到申請後 **15 個工作日**內回覆。複雜案件得延長至 30 個工作日，並通知您延長原因。

**身份驗證**：為保護您的個資安全，行使上述權利前須驗證帳號身份（以帳號登入之 Email 申請）。

**費用**：行使個資權利原則上免費。若申請明顯重複或無理，本公司保留收取合理行政費用之權利。

**申訴管道**：若您認為本公司違反個資法，除向本公司申訴外，亦可向個人資料保護主管機關（個人資料保護委員會）提出申訴。

## 十一、未成年人之保護

本服務主要面向專業知識工作者，**不針對未滿 18 歲之未成年人提供服務**。

若您未滿 18 歲，請勿使用本服務或提供任何個人資料。若本公司發現不慎蒐集未成年人之個人資料，將立即刪除相關資料，並通知其法定代理人（如聯絡方式可取得）。
